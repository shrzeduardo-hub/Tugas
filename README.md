# 🌐 Portfolio Website – Sahrul Reza Eduardo

Website portfolio pribadi yang dibuat menggunakan **HTML, CSS, dan JavaScript** untuk menampilkan profil, keahlian, proyek, dan informasi kontak.

## 📖 Deskripsi

Project ini merupakan website portfolio sederhana yang digunakan untuk memperkenalkan diri sebagai **Full Stack Web Developer dan IT Student**.
Website ini menampilkan informasi tentang profil, skill, portfolio proyek, serta kontak yang dapat dihubungi.

Website dirancang dengan desain **modern, responsif, dan interaktif** sehingga dapat diakses dengan baik di berbagai perangkat seperti desktop, tablet, maupun smartphone.

---

## 🚀 Fitur Website

* **Header Profile**

  * Nama dan deskripsi singkat developer
  * Tampilan hero section modern

* **Navigation Bar**

  * Navigasi menuju setiap section halaman
  * Smooth scrolling

* **About Section**

  * Informasi tentang diri developer
  * Foto profil

* **Skills Section**

  * Menampilkan keahlian seperti:

    * HTML
    * CSS
    * JavaScript
    * Git

* **Portfolio Section**

  * Menampilkan beberapa contoh proyek:

    * Website Toko Online
    * Aplikasi To-Do List
    * Landing Page Produk

* **Contact Section**

  * Form kontak
  * Informasi email, telepon, dan LinkedIn

* **Interaktif JavaScript**

  * Smooth scrolling navigation
  * Validasi form kontak
  * Visitor counter menggunakan LocalStorage
  * Back to top button
  * Animasi fade-in

---

## 🛠️ Teknologi yang Digunakan

* **HTML5** – Struktur halaman website
* **CSS3** – Styling dan desain tampilan
* **JavaScript** – Interaktivitas website
* **Font Awesome** – Icon website
* **Google Fonts (Poppins)** – Typography
